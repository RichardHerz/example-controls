==================== NOTES ON JAVASCRIPT ===================== 

Consider using existing plotting scripts and not extracting parts into an example file... 

Arrays that hold data for x-y data plotting are stripData and profileData and are initialized in process_plot_info.js by calling initialization function in process_plotter.js (initializing in process_units.js did not work - see comment there) 

The data in the stripData and profileData arrays are updated by the process units in process_units.js. 

The strip and profile plots only differ in that a complete new data set is filled at each time for profile plots, whereas only one new data point pushed in and old out for strip plots, and these updates are done in process_units.js. 

The plots are defined in process_plot_info.js. 

The plots are plotted by call in process_main.js and executed by functions in process_plotter.js. 

FOR SPACE-TIME COLOR PLOT

Array spaceTimeData is defined as a GLOBAL and initialized in process_spacetime.js. Array spaceTimeData is updated in process_units.js. Array spaceTimeData is converted to color values and plotted in process_spacetime.js in function plotSpaceTimePlot.
