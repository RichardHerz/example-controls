Scrolling colormap:

Click Run button to start plotting...

This is an example of graphics using the HTML canvas element. In this scrolling space-time colormap, time is represented on the horizontal axis, and space on the vertical axis. As time proceeds, a record of a sine wave that is scrolling down in the vertical direction is shown. The value of the sine wave at each time and vertical position is represented as a color value. Random changes in the sine wave's value are made so that you know the graphic values are being computed and that we are not simply showing you an animated gif! :)
